# Task Manager - Laravel

Projeto de gerenciamento de tarefas desenvolvido em Laravel.

## Funcionalidades
- CRUD de tarefas
- Marcar como concluída
- Filtro (todas, pendentes, concluídas)
- Prioridade (low, medium, high)

## Tecnologias
- Laravel
- PHP
- MySQL
- Bootstrap

## Como rodar o projeto

1° Instalar o PHP e o Composer
2° Composer global require laravel/installer
3° Criar uma pasta
4° cd nome da pasta criada
5° npm install && npm run build
6° composer run dev
7° laravel new to-do-list
8° Conectar em algum banco de dadas no .env (Recomendo MySQL)
9° php artisan migrate
10° php artisan serve
